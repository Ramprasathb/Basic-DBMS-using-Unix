This is a simple database compiler where the user can create a database , create a table , select a database , insert values into the table..
He will be able to select all the columns from the table , or select particular columns alone from the database..
He will also be able to specify conditions for selecting the records from the table..
A database has to be selected in order to manipulate tables.

Syntax :

Create table :
             create table tablename
Insert values :
             insert into table tablename
create database :
             create database databaseName
select database :
	     select database databaseName
select :
       select all from tableName
       select all from tableName where condition1 = value1 and condition2 = value2
       select column1 column2 .. columnN from tablename 
       select column1 column2 .. columnN from tablename where condition1 = value1 and condition2 = value2
